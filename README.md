# lingq-swedish-auto-upload

Imports some Swedish resources with audio to the [LingQ](https://www.lingq.com/en/) course you specified.

## Deploy

To run locally without launching the web server, don't forget to rename `.env.example` to `.env` and set the values.

You can also deploy this onto Heroku.
